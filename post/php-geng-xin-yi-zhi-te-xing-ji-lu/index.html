PHP更新移植特性记录

· 2022-05-14 · # php
从 PHP 5.6.x 移植到 PHP 7.0.x
  1. 标量类型声明；
  2. 返回值类型声明；
  3. null合并运算符：
  4. 太空船操作符（组合比较符）等；
从 PHP 7.0.x 移植到 PHP 7.1.x ¶

返回值可为空（Nullable）类型；

  1. 返回值可为void类型；
  2. 短数组语法（[]）作为list()语法的备选项；
  3. 类常量可见性；
  4. 多异常捕获处理；
  5. list()现在支持键名；
  6. 支持为负的字符串偏移量等；
从 PHP 7.1.x 移植到 PHP 7.2.x ¶
  1. 返回值可为对象类型；
  2. 允许重写抽象方法(Abstract method)等；
从 PHP 7.2.x 移植到 PHP 7.3.x ¶
从 PHP 7.3.x 移植到 PHP 7.4.x ¶
  1. 属性添加限定类型;
  2. 空合并运算符赋值等;
从 PHP 7.4.x 移植到 PHP 8.0.x ¶
  1. 命名参数；
  2. 注解（Attributes）等;
从 PHP 8.0.x 移植到 PHP 8.1.x ¶

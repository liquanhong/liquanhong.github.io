常用的docker命令

· 2021-07-04 · # docker

Docker镜像常用命令

下载镜像的命令：docker image pull
列出镜像列表: docker images
删除镜像: docker image rm

Docker容器常用命令

查看容器：docker ps -a
查看运行的容器：docker ps
删除容器(-f强制删除运行中容器)：docker rm -f 1e560fca3906
进入容器：docker exec -it xxx /bin/sh
重启容器：docker restart 容器ID
停止容器：docker stop 容器ID
清理所有处于终止状态的容器：docker prune

Docker Compose常用命令

构建（重新构建）项目中的服务容器：docker-compose build 容器名字（可无）
用于部署一个 Compose 应用：docker-compose up -d 容器名字（可无）
启动Compose 应用相关的所有容器：docker-compose start
停止并删除运行中的 Compose 应用：docker-compose down
停止 Compose 应用相关的所有容器：docker-compose stop
重启已停止的 Compose 应用：docker-compose restart
解析所有变量：docker-compose config

参考：https://www.runoob.com/docker/docker-container-usage.html
